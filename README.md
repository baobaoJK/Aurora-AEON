## 开发环境

网页制作工具：VSCode

JS 插件：jsmediatags.js



## 简介

一个 本地 JS 音乐播发器，默认列表循环播放，支持单曲循环，功能完善。

使用 jsmediatags.js 插件实现获取 mp3 文件图片，名称，作者信息，仅支持 mp3 格式文件。

软件界面仿网易云音乐

主页界面仿 兰空 VOEZ 风格



素材均来源于网络，切勿用于商业用途，此项目仅供学习参考，个人行为后果自负。
